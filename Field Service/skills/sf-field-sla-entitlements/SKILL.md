# sf-field-sla-entitlements

SLA, entitlement, and milestone strategy for onsite service commitments.

Use when users need response/resolve commitments, milestone design, escalation policy, and business-hours alignment for Field Service operations.

TRIGGER when:
- User asks for premium vs standard service commitments.
- User asks how to enforce response windows and breach actions.
- User asks how to model service contracts with field execution.

DO NOT TRIGGER when:
- Task is focused on work-order status lifecycle (use `sf-field-work-order-lifecycle`).
- Task is focused on optimization/capacity tuning (use `sf-field-optimization-capacity`).

Primary outcomes:
- Milestone model tied to appointment lifecycle.
- Escalation and breach remediation playbook.
- KPI framework for contractual performance.
